# ChotkiyModal

Легковесный и гибкий класс на чистом JavaScript для создания модальных окон. Проект разработан с акцентом на доступность (A11y) и удобство интеграции в современные веб-интерфейсы.

> **Важно:** Данный проект является демонстрацией навыков в области фронтенд-разработки. Он реализует логику доступности (ловушка фокуса, `inert`, возврат фокуса) без использования тяжелых сторонних библиотек.

Пример - https://sirflexsir.github.io/ChotkiyModal/demo/index.html

---

## Особенности

- **Keyboard Navigation:** Полная поддержка управления с клавиатуры (Esc для закрытия, Tab для навигации).
- **Focus Trapping:** Умная «ловушка» фокуса — фокус не покидает модальное окно, пока оно активно.
- **Modern A11y:** Использование атрибута `inert` для корректной работы скринридеров.
- **Flexible Config:** Индивидуальная настройка каждого экземпляра (подложка, закрытие по клику, ловушка фокуса).
- **Singleton Shadow:** Единый элемент подложки для всех окон, исключающий дублирование кода в DOM.

---

## Демонстрация навыков

Данный код написан для демонстрации следующих компетенций:

- **Управление состоянием:** Отслеживание активной модалки через статическое свойство `Modal.activeModal`.
- **Работа с DOM:** Динамическое создание элементов (оверлей), поиск по data-атрибутам, управление классами и атрибутами.
- **Событийная модель:** Делегирование событий, обработка клавиатурных событий, предотвращение всплытия.
- **Accessibility:** Глубокое понимание `aria-hidden` (через `inert`), управления `tabindex` и нюансов работы с фреймами.
- **Работа с фокусом:** Реализация ловушки фокуса.

## Использование (Usage)

Для работы требуется следующая структура и атрибуты:

### Обязательные атрибуты:

1.  **Кнопка открытия:** должна иметь атрибут `data-js-open-modal` со значением id модалки (например, `#myModal`).
2.  **Модальное окно:** должно иметь `id`, совпадающий с селектором в кнопке открытия.
3.  **Кнопка(и) закрытия:** должна(ы) иметь атрибут `data-js-close-modal`.

### 1. Разметка (HTML)

```html
<button data-js-open-modal="#myModal">Открыть окно</button>

<div class="chotkiyModal close" id="myModal">
	<div class="chotkiyModal__window">
		<div class="chotkiyModal__content">
			<button data-js-close-modal>Закрыть</button>
			<h2>Заголовок</h2>
			<p>Текст...</p>
		</div>
	</div>
</div>
```

```js
import Modal from './modal.js'

// Простая инициализация
new Modal({
	idModal: '#myModal',
})

// Кастомная конфигурация
new Modal({
	idModal: '#anotherModal',
	closeOnEscKey: false, // Отключить закрытие по Escape
	trapFocus: false, // Отключить ловушку фокуса
	firstFocus: false, // Не возвращать фокус при закрытии
	closeModalClick: false, // Не закрывать по клику на оверлей
	shadow: false, // Не показывать оверлей
})
```
